### 실행흐름

0. 라파 <-> 폰 블루투스 페어링 서로 등록: 수동으로 했음, 후에 작업해야 함
1. 앱 실행
2. 라즈베리파이 블루투스 연결
3. SSID, PW 입력
4. 현재 라파가 네트워크에 연결되었는지 어느 와이파이에 연결되었는지 확인가능
