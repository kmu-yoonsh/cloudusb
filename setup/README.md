# Setup Raspberry Pi Zero W

### usage:
	sh setup.sh

### It downloads the followings.
	1. vim
	2. rpi-source
	3. bc
	4. libncurses5-dev
	5. /piusb.bin
	6. python-pip
	7. google-api-python-client

### It also sets up the followings.
	1. /etc/default/keyboard
	2. /boot/config.txt
	3. /etc/modules
	4. ~/.vimrc
